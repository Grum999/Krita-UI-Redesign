Krita UI Redesign Manual

Just a simple plugin to improve the overall Krita UI and UX!

Features collapsible toolbar and tool options, with customizable shortcuts. Default is "T" for the toolbox, and "N" for the Tool Options.

Features and How-to-use:

  • After activating the plugin, a new tab in the top menu bar will appear;
  • You can choose which parts of the plugin to use;

Hope you enjoy this plugin, and feel free to post your feedback over on Krita Artists!
